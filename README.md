راهنمای راه‌اندازی اتصال آزاد (نسخه نهایی و مستقل)

بااین روش، شما با استفاده از اکانت جیمیل خودتان، یک مسیر امن و اختصاصی برای دسترسی به اینترنت آزاد می‌سازید. این کار کاملاً رایگان است و نیازی به دانش فنی ندارد.

مراحل به دو بخش تقسیم می‌شود: یک‌بار تنظیمات در گوگل و استفاده از برنامه در کامپیوتر.

🛠️ بخش اول: ساخت مسیر اختصاصی روی گوگل (فقط یک‌بار)

۱. مرورگر خود را باز کنید، وارد اکانت جیمیل خود شوید و به این آدرس بروید: script.google.com
۲. روی دکمه New project (پروژه جدید) کلیک کنید.
۳. کدهای پیش‌فرض را پاک کنید و کدهایی که در فایل Code.gs برایتان فرستاده‌ام را آنجا پیست (Paste) کنید.

۴. در خط پنجم کد، این عبارت را پیدا کنید:

const AUTH_KEY = "xxxxxxxxxxxxx";
به جایxxxxxxxxxxxx، یک رمز دلخواه و انگلیسی برای خودتان بنویسید و آن را به خاطر بسپارید

۵. در بالای صفحه، سمت راست، روی دکمه آبی رنگ Deploy کلیک کرده و New deployment را انتخاب کنید.
۶. روی علامت چرخ‌دنده (⚙️) کلیک کنید و Web app را انتخاب کنید.
۷. تنظیمات را دقیقاً این‌طور قرار دهید:

بخش Execute as: روی گزینه Me * بخش Who has access: روی گزینه Anyone ۸. دکمه Deploy را بزنید. (اگر گوگل از شما تاییدیه خواست، اکانتتان را انتخاب کرده و Allow را بزنید).
۹. در پایان، یک کد طولانی به نام Deployment ID به شما داده می‌شود. آن را کپی کنید.

💻 بخش دوم: راه‌اندازی در ویندوز
۱. فایل VPN.exe را دانلود کنید و در یک پوشه جدید قرار دهید.
۲. (بسیار مهم): روی فایل VPN.exe کلیک راست کنید و گزینه Run as administrator را بزنید.

⚠️ نکته امنیتی ویندوز: چون این برنامه شخصی است، ممکن است ویندوز یک صفحه آبی‌رنگ (SmartScreen) نشان دهد. اصلاً نگران نباشید! روی نوشته More info کلیک کنید و دکمه Run anyway را بزنید.

۳. وقتی برنامه باز شد، Deployment ID و رمزی که در بخش قبل ساختید را در کادرهای مربوطه وارد کنید.
۴. روی دکمه Connect کلیک کنید.

⚠️ نکته نصب گواهینامه: برای بار اول، ویندوز یک پیام باز می‌کند و می‌پرسد آیا اجازه می‌دهید یک Certificate نصب شود؟ حتماً Yes را بزنید تا سایت‌ها امن باز شوند.

✅ تمام شد! اینترنت شما آزاد است. (هر زمان کارتان تمام شد، حتماً دکمه Disconnect را بزنید تا اینترنت سیستم به حالت عادی برگردد).

🦊 نکته مهم فقط برای کاربران مرورگر فایرفاکس (Firefox)
اگر از کروم (Chrome) یا اج (Edge) استفاده می‌کنید، نیازی به این مرحله ندارید. اما اگر فایرفاکس دارید، برای جلوگیری از خطای امنیتی سایت‌ها، این یک کار کوچک را انجام دهید:

۱. در فایرفاکس به تنظیمات (Settings) بروید.
۲. از منوی سمت چپ Privacy & Security را انتخاب کنید و به پایین صفحه بخش Certificates بروید.
۳. روی دکمه View Certificates کلیک کنید.
۴. تب Authorities را انتخاب کرده و دکمه Import را بزنید.
۵. به همان پوشه‌ای بروید که VPN.exe قرار دارد. می‌بینید که یک پوشه جدید به نام ca ساخته شده است. وارد آن شوید و فایل ca.crt را انتخاب کنید.
۶. تیک گزینه Trust this CA to identify websites را بزنید و OK کنید.

مرورگر فایرفاکس را ببندید و دوباره باز کنید. حالا همه سایت‌ها بدون مشکل باز می‌شوند!
